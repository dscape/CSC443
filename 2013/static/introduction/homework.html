Programming environment

  1. We assume that you work in a Linux working environment.

  2. The main programming language for the assignments will be C / C++. For the final project, you are free to use your own language(s) of choice.

  3. Marking of your assignments will be automated. We will rely on Make pin to compile your code. It is important that you are familiar with Make.

  4. You will be utilizing the leveldb library as a B+ tree implementation.

    • leveldbpin can be downloaded from: Google code

    • Make sure you can compile the sample codes using this Makefile

      • Creating and writing to leveldb: write.cc
      • Opening and reading from leveldb: read.cc
comment

Examine the Makefile, and see that it requires you to properly define environment variable: $LEVELDB_DIR

  • Compile the binaries using: make all

  • Run the tests using: make test

  • Cleanup the build using: make clean

References

1 Make: a utility that automatically builds executable programs and libraries. See Wikipedia.
2 LevelDB: a fast key-value storage library written at Google that provides an ordered mapping from string keys to string values. Available at Google Code.
